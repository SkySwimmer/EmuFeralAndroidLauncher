Installing EmuFeral on Mobile

Welcome to the installation wizard of EmuFeral Online for Android!

Here, you will find how to get a modified client for our Fer.al server emulator.
First, you will need to download the Easy Android Application Modification Toolset by the AerialWorks Software Foundation
(organization that develops EmuFeral), you can download it here from our download server.

In order to run the modification toolset jar, you will need to install Java 15 or later, we recommend
Eclipse Adoptium for this. (not our software, just recommended)

When you have all that, you can open the modification toolset

After opening the modification tools, enter the following in the Automatic Modification Server Path box: <server ip >/api/mod.info
Click 'build modified app' and wait for it to complete.

Finally, install the modified application

After the process completes, the launcher will prompt to remove the old fer.al client, allow this.
After the old client is deleted, it will ask to install the modified client, after you install it, the game should start.

Thats all you need to do, enjoy!
